# Portfolio Wafaa Jemel

Ce projet a été construit avec **React** et **Vite**.

## Comment lancer le projet localement

1. **Installer Node.js** : Téléchargez et installez Node.js depuis [nodejs.org](https://nodejs.org/).
2. **Ouvrir un terminal** : Allez dans le dossier du projet.
3. **Installer les dépendances** :
   ```bash
   npm install
   ```
4. **Lancer le serveur de développement** :
   ```bash
   npm run dev
   ```
5. **Voir le site** : Ouvrez votre navigateur sur `http://localhost:3000`.

## Comment déployer en ligne (Gratuit)

Pour que les recruteurs puissent voir votre site, vous pouvez le mettre en ligne gratuitement en 2 minutes sur **Vercel** ou **Netlify** :
1. Glissez-déposez simplement le dossier du projet sur leur interface.
2. Ils s'occuperont de tout et vous donneront un lien (ex: `wafaa-jemel.vercel.app`).
